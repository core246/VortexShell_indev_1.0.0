# VortexShell_indev_1.0.0

CoreShell IDE — это легковесная портативная среда разработки для Python на базе клиент-серверной архитектуры. Программа разделяет графический интерфейс и среду выполнения, организуя взаимодействие с фоновым интерпретатором через потоки ввода-вывода. Это изолирует пользовательский код, исключает зависание оболочки и экономит оперативную память.
